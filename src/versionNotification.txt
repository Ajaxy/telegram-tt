**LisyanGramm Release Notes**

Over the past year, many new features and useful optimisations have been added. Here is a brief overview of the developments that have taken place in LisyanGramm during those busy months.

✏️ __General__
• A huge number of design fixes across the app.
• Preview chats and mark them as read.
• Video messages can now be transcribed.
• Track your tasks with the help of [checklists](https://telegram.org/blog/checklists-suggested-posts/#checklists).
• Folder tags are now supported with the help of GitHub contributors.
• [Send photos in HD quality](https://telegram.org/blog/direct-to-channel-trim-voice-and-more#send-photos-in-higher-quality).
• [Global search of public posts](https://telegram.org/blog/post-search-story-albums-and-more#public-post-search).
• Support for channels with enabled auto-translate has been added.

👤 __Profile__
• Full redesign of the profile page.
• [Explore other people's story albums](https://telegram.org/blog/post-search-story-albums-and-more#story-albums).
• Keep an eye on all your accounts, just like on mobile and desktop clients, with the multi-account feature.

There are so many other bugfixes and improvements that it would take forever to describe them all! We're continually working on adding new features and improving the user experience.
